# Gapsi E-commerce

Aplicación e-commerce desarrollada con Angular 21, standalone components y signals. Es una PWA instalable.

**Demo en vivo:** https://gapsi-test.maxalva.dev/

## Clonar el repositorio

```bash
git clone https://github.com/alvag/gapsi-test.git
cd gapsi-test
npm install
```

## Servidor de desarrollo

```bash
ng serve
```

Abre el navegador en `http://localhost:4200/`. La aplicación se recarga automáticamente al modificar archivos.

## Compilar para producción

```bash
ng build
```

Los archivos compilados se guardan en el directorio `dist/`.

## PWA

La aplicación es una Progressive Web App instalable. Incluye service worker, manifest y soporte offline.
